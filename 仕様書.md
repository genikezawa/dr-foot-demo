## 1. サービス概要

iPhoneのカメラとLiDARセンサーを使ってユーザーの足を3Dスキャンし、正確な寸法と足型診断をもとにオーダーメイド革靴を購入できるサービス。

**コンセプト**
> 店に行かずに、あなたの足に合うオーダー革靴を作れる

**ターゲット顧客**
- 30〜50代男性
- 仕事で革靴を履く
- 既製靴の痛みに悩んでいる（甲高・幅広・左右差・外反母趾気味）
- オーダーメイドに興味はあるが百貨店に行くのは面倒
- 4万円程度を支払える

**収益モデル**
- 提携靴業者からマージンを受け取る（靴1足につき1万円）
- 靴のオーダーメイド平均価格: 40,000円前後

## 4. 開発フェーズ

### Phase 1（先行リリース）— 現在進行中

**目的**: FootScannerアプリを単体でApp Storeにリリースし、提携靴業者経由で実績を作る。

**ユーザーフロー**
```
LP（GitHub Pages）
  ↓ App Storeへ誘導
FootScannerアプリをダウンロード
  ↓
左右の足をLiDARスキャン（約10分）
  ↓
アプリ内で簡易足型診断を表示
  ↓
「靴を注文する」ボタンをタップ

```

## 5. FootScannerアプリ仕様

### 現在の実装

| ファイル | 役割 |
|---|---|
| `ContentView.swift` | ホーム画面（右足・左足スキャンボタン、プレビュー、エクスポート） |
| `ScanView.swift` | LiDARスキャン画面（ObjectCapture、音声ガイド付き） |
| `ModelPreviewView.swift` | 3Dモデルプレビュー |
| `ZipExporter.swift` | OBJ/USDZ/MTLをZIP化 |
| `Item.swift` | SwiftDataモデル（FootSide, modelURL） |
| `ARCapabilities.swift` | LiDAR対応チェック |
| `ScanPaths.swift` | ファイルパス管理 |

### スキャン手順（ScanView）
1. preparingStep1〜4: 音声ガイドで準備（椅子に座る → 足を所定位置に置く）
2. framing: AR空間でオブジェクト検出
3. scanning: ObjectCaptureでスキャン
4. processing: OBJ/USDZ生成
5. 完了後: ContentViewでプレビュー表示・エクスポート可能

### 出力ファイル
- `right_foot.usdz` / `right_foot.obj` / `right_foot.mtl`
- `left_foot.usdz` / `left_foot.obj` / `left_foot.mtl`
- エクスポート時にZIPに圧縮

### 簡易寸法計算の方針（Phase 1）
- OBJファイルの頂点座標からバウンディングボックスを計算
- ObjectCaptureはLiDAR由来の実世界スケール（メートル単位）で出力するため、スケール補正は原則不要
- 足の向きはスキャン時のARガイドでほぼ揃っているが完全固定ではない
- 最長軸 = 足長、中間軸 = 足幅、最短軸 = 甲高 として近似
- 精度目安: ±3〜5mm（Phase 1では許容範囲とする）
